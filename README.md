# 🛍️ Productify – Full-Stack Product Store

> A modern, scalable full-stack product management platform built with TypeScript and PostgreSQL — designed for performance, clean architecture, and beautiful user experience.

🌐 **Live Demo:**  
https://productify-full-stack-product-store.up.railway.app/

---

## ✨ Overview

Productify is a cloud-deployed full-stack web application that allows users to explore, create, manage, and interact with products in a seamless and intuitive environment.

From secure authentication to real-time data synchronization, the application demonstrates complete end-to-end integration between frontend and backend systems.

---

# 🚀 Key Highlights

🔐 **Secure Authentication**  
User authentication and authorization powered by Clerk.

⚙️ **Robust REST API**  
Backend built with Node.js & Express handling full CRUD functionality.

🧬 **Type-Safe Backend**  
Built with TypeScript and Drizzle ORM for safer, maintainable database interactions.

🗄️ **PostgreSQL Database**  
Relational data management for users, products, and comments.

⚡ **Efficient Data Fetching**  
TanStack Query enables caching, optimized fetching, and smooth UI updates.

🎨 **Modern UI/UX**  
Styled using Tailwind CSS & DaisyUI for a clean, responsive design.

☁️ **Cloud Deployment**  
Fully deployed and connected to a cloud-hosted PostgreSQL database.

# 🧠 Architecture Overview

Productify follows a structured full-stack architecture:

Frontend (React)  
⬇  
REST API (Express + TypeScript)  
⬇  
Drizzle ORM  
⬇  
PostgreSQL Database (Neon)  

This ensures clean separation of concerns, scalability, and maintainability.
